# almond-tree# Almond Tree — GitHub Pages Website

This folder is ready to upload to GitHub Pages.

## Publish it for free

1. Go to GitHub and create a free account.
2. Click **New repository**.
3. Name it, for example: `almond-tree-links`
4. Select **Public** and click **Create repository**.
5. Click **Add file → Upload files**.
6. Upload `index.html`.
7. Open **Settings → Pages**.
8. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
9. Click **Save**.

After publishing, your link will look like:

`https://YOUR-GITHUB-USERNAME.github.io/almond-tree-links/`

Use that final URL to create one QR code.

## Editing later

Open `index.html` in GitHub, click the pencil icon, change the text or links, and click **Commit changes**.

The printed QR code will continue working as long as the GitHub Pages URL stays the same.
